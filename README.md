## :information_source: How To Use

To clone and run this application, you'll need [Git](https://git-scm.com), [Node.js v10.16][nodejs] or higher + [Yarn v1.13][yarn] or higher installed on your computer and the [barberapp API](https://github.com/GabrielFagundes/barberapp-backend). From your command line:

```bash
# Clone this repository
$ git clone https://github.com/gabrelfagundes/barberapp-web

# Go into the repository
$ cd barberapp-web

# Install dependencies
$ yarn install

# Run the app
$ yarn start
```
